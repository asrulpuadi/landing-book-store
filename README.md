<h1>Landing Page Book Store Terintegrasi Saweria</h1>

<h3>Deskripsi Proyek</h3>
<p>Proyek ini merupakan tugas Analisis dan Desain Sistem Informasi (SI) dengan tema Landing Page Book Store yang terintegrasi dengan platform donasi Saweria. Sistem ini dirancang untuk mempromosikan dan menjual buku secara online sekaligus memberikan kemudahan bagi pengunjung untuk memberikan dukungan/donasi melalui Saweria.</p>

<p>Landing page berfungsi sebagai media informasi, promosi, dan interaksi awal antara pengguna dengan Book Store. Integrasi Saweria memungkinkan model monetisasi tambahan berupa donasi atau dukungan langsung dari pengunjung.</p>
